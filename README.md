# 模板例子，基于 Vue 3.0

## Project setup
```
yarn / npm i
```

### 开发测试
```
yarn serve / npm run serve
```

### 开发生产
```
yarn serve:prod / npm run serve:prod
```

### 打包测试
```
yarn build:test / npm run build:test
```

### 打包生产
```
yarn build:prod / npm run build:prod
```

### Lints and fixes files
```
yarn lint
```
